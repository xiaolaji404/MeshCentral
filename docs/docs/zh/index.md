# MeshCentral 文档

## 关于

MeshCentral 是一个功能齐全的计算机管理网站。借助 MeshCentral，您可以运行自己的 Web 服务器，远程管理和控制本地网络或互联网上的任何计算机。启动服务器后，创建设备组，并在每台要管理的计算机上下载并安装代理。一分钟后，新计算机就会出现在网站上，您可以对其进行控制。MeshCentral 包含完整的基于 Web 的远程桌面、终端和文件管理功能。

想要获取更多信息, [请访问 MeshCentral.com](https://meshcentral.com)

## 💬 反馈

如果您遇到问题或有任何改进产品的建议，可以提交 [GitHub Issue](https://github.com/Ylianst/MeshCentral/issues/)。

如果您要提交问题报告，请包含以下信息：

* 你正在使用的软件版本
> 例如：1.1.46
* 你的操作系统以及版本
> 例如：Debian 12
* 您为自行解决问题而采取的任何故障排除措施。
> 例如：重新安装了MeshCentral（包括操作系统）
* 其他类似的信息
> 例如：其他的Github Issues
* 实际的输出
* 预期的输出

如果您在使用以下其他产品时遇到问题，请在相应的 issue 页面提交报告：
